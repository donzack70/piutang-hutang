🔥 1. Hubungkan Firebase
Config disimpan di browser ini — cukup isi sekali.
Belum punya project Firebase? Klik untuk panduan setup (sekali saja)
1Buka console.firebase.google.com → Add project (nama bebas, mis. warna-indah). Google Analytics boleh dimatikan.
2Di menu kiri: Build → Firestore Database → Create database. Pilih lokasi asia-southeast2 (Jakarta), mode production.
3Masih di Firestore, tab Rules — ganti isinya dengan aturan di bawah lalu Publish (hanya user yang login yang bisa baca/tulis):
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
4Build → Authentication → Get started → Email/Password → Enable. Lalu di tab Users → Add user, buat email & password untuk toko.
5Klik ikon ⚙ Project settings → scroll ke Your apps → pilih ikon web </> → daftarkan app (nama bebas) → copy bagian firebaseConfig = { ... } dan paste di kotak di atas.
🔐 2. Masuk
Akun bisa juga dibuat lewat Firebase Console → Authentication → Users.
